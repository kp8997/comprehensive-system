POJO - Old Plain Java Object:

    - Generally has instance fields.

    - Usually has constructor, getter, setter, equals(), hashCode(), toString() methods.

    - Database framework often use them such as Entity, DTO, Configuration Object Spring Boot.

    - Sometime can be called JavaBean.
        - POJO (Plain Old Java Object)
            + Any standard Java object
            + No framework dependencies
        - JavaBean
            + Must have a public no-arg constructor
            + Must have private fields with getters/setters
            + Must be Serializable
